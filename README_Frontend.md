# 📅 Schedule Pro - Employee Scheduling System

## 🚀 Frontend Documentation

A complete Employee Scheduling System built with React that supports three user roles: **Admin**, **Manager**, and **Employee**. This is the frontend application for the Schedule Pro project.

\---



## 🎯 Project Overview

**Schedule Pro** is a comprehensive employee scheduling and project management system. It allows organizations to:

* Manage employee shifts and schedules
* Track leaves and attendance
* Manage projects and tasks
* Handle shift swap requests
* Generate reports and analytics
* Send announcements to employees

\---

## ✨ Features

### 👑 Admin Features

* **User Management**: Create, edit, delete users (Admin, Manager, Employee)
* **Department Management**: Create and manage departments
* **Company Schedule**: View all schedules across the company
* **Leave Overview**: View and manage all leave requests
* **Swap Overview**: View and manage all swap requests
* **Project Overview**: View all projects company-wide
* **Task Overview**: View and manage all tasks
* **Reports**: Generate system-wide reports
* **Announcements**: Send announcements to all users

### 📊 Manager Features

* **Team Management**: View team members, invite new members
* **Team Schedule**: Create and manage team schedules
* **Leave Approvals**: Approve or reject leave requests
* **Swap Approvals**: Approve or reject shift swap requests
* **Project Management**: Create and manage projects
* **Task Management**: Create tasks, approve submissions
* **Reports**: View team performance reports

### 👤 Employee Features

* **Dashboard**: View personal statistics and upcoming schedules
* **My Schedule**: View weekly/monthly work schedule
* **Swap Request**: Request shift swaps with other employees
* **Leave Requests**: Apply for leaves, view balance
* **My Projects**: View assigned projects
* **My Tasks**: View tasks, update progress, submit work
* **My Progress**: Track weekly work progress
* **My Profile**: View and edit profile, upload photo
* **Invitations**: Accept team invitations
